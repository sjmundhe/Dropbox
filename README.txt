
Install ReactJS

On the terminal: npm install -g create-react-app

Follow the steps on the terminal to start the servers:

Back-end server
	1. cd nodelogin
	2. npm install
	3. npm start

Front-end server
	1. cd reactlogin
	2. npm install
	3. npm start
Kafka-backend server
	1. cd Kafka-backend
	2. npm install
	3. npm start
